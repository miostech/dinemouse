# Dine Mouse - Landing Page Premium

Landing page institucional premium para Dine Mouse, serviço independente de concierge gastronômico especializado em reservas de restaurantes nos parques temáticos de Orlando e Paris.

## 🎨 Design

### Paleta de Cores
- **Azul Profundo**: `#0B1C2D`
- **Roxo Mágico**: `#4B3F72`
- **Dourado Suave**: `#E6C77A`
- **Branco**: `#FFFFFF`

### Características Visuais
- ✨ Gradientes noturnos elegantes
- 🌟 Partículas animadas (estrelas/brilhos)
- 🎭 Micro-animações suaves
- 📱 Design mobile-first e responsivo
- 🎯 Tipografia elegante (Inter + Playfair Display)

## 📋 Estrutura do Site

1. **Hero Section** - Impacto emocional com partículas animadas
2. **Problema** - Apresentação clara das dores do cliente
3. **Solução** - Como a Dine Mouse resolve o problema
4. **Parques Atendidos** - Orlando e Paris
5. **Como Funciona** - Processo passo a passo
6. **Prova de Valor** - Benefícios principais
7. **CTA Final** - Conversão com WhatsApp e formulário
8. **Footer** - Informações legais e links

## 📝 Personalização

### WhatsApp
Edite o número de telefone no arquivo `script.js`:
```javascript
const phoneNumber = '5511999999999'; // Seu número aqui
```

### Formulário
O formulário pode ser integrado com:
- Backend próprio
- Serviços de email (EmailJS, Formspree, etc.)
- APIs de terceiros

### Cores
As cores podem ser ajustadas nas variáveis CSS em `styles.css`:
```css
:root {
    --azul-profundo: #0B1C2D;
    --roxo-magico: #4B3F72;
    --dourado-suave: #E6C77A;
    /* ... */
}
```
